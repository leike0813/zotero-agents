# Zotero Agents

エージェントスキルを実行するためのZoteroプラグイン。

![Zotero Agents 研究ワークベンチポスター](/img/poster.png)

## Zotero Agentsとは

Zotero Agentsは、Zoteroをインテリジェントエージェント時代に向けたパーソナルリサーチワークベンチに変える。文献ライブラリ、エージェントバックエンド、Workflow、ナレッジグラフ、外部ツールを接続し、文献分析を一回限りのQ&Aから、持続可能で監査可能かつ拡張可能な研究プロセスへと転換する。

能力の第一層は**プラグイン可能なWorkflow**である。研究者は複雑な文献タスクを再利用可能なプロセスに分割できる。論文の解析、深読、引用分析、タグの正規化、文献検索、トピックの統合、レビュー資料の生成などである。Workflowは異なるエージェントやサービスバックエンドに接続し、エージェントの長文コンテキスト理解、ツール呼び出し、マルチステップ推論を活用して、 otherwise 反復手作業が必要だった文献管理・分析ワークフローを自動化し、研究ニーズの進化に応じて拡張できる。

第二層は**アシスタントサイドバー**である。コーディングエージェントスタイルの対話型インターフェースを提供し、ACPプロトコルを介した各種エージェントバックエンドへの接続や、Skill-Runnerバックエンドを介した特定のWorkflowの実行をサポートする。現在のアイテム、選択した文献、またはライブラリ全体に基づいて、エージェントに質問させ、論文を分析させ、関連研究を検索させ、参考文献をライブラリに追加させ、長時間実行されるタスク中の対話、確認、修正、進捗追跡を継続できる。

第三層は**Synthesis Workbench**である。ライブラリレベルの長期的な知識構築を対象とし、個別の論文分析から生成されたダイジェスト、参考文献、引用セマンティクス、タグ、概念、トピック関係を統一的なナレッジプラットフォームに統合する。研究者はここで参考文献ネットワークを管理し、引用マッチをレビューし、引用グラフを探求し、トピックを中心に文献を整理し、トピック統合を使用して研究領域の基盤文献、最先端の研究、主要な議論、方法論的相違、カバレッジのギャップ、将来の方向性を整理できる。その目標は、広範な読書をレビュー、学位論文提案、論文の導入、研究ロードマップ設計に適した構造化された資料に変換することである。

第四層は**Host Bridge**である。`zotero-bridge` CLIとMCPサービスを介して、外部エージェントがZoteroライブラリと直接対話できる。文献コンテキストの読み取り、アイテムの検索、新しい参考文献の追加、分析タスクの呼び出し、構造化された結果の書き戻しが可能である。OpenClawやHermesなどのエージェントWorkflowにより、文献検索、フィルタリング、分析、要約、レビュー起草を委任でき、長時間実行される研究タスクがバックグラウンドで継続的に進行する。

Zotero Agentsの中核的な価値は、Zoteroライブラリをエージェントが真に作業できる研究環境にすることである。すべての読書、分析、レビュー、執筆準備のステップが、次の研究フェーズの知識として蓄積される。

> **対応Zoteroバージョン**: 本プラグインはZotero 7およびZotero 9をサポートする。主な開発とテストはZotero 9で行われている。Zotero 8は理論上完全にサポートされている（プラグインフレームワークは8/9間で変更なし）。Zotero 7も理論上動作するはずだが、十分にテストされてはいない。今後のメンテナンスはZotero 9に注力する。Zotero 7で問題が発生した場合は[Issues](https://github.com/leike0813/zotero-agents/issues)で報告されたい。

:::tip ヒント
本プラグインには**組み込みのビジネスロジックはない**。すべてのWorkflowは、プラグインのインストール後にユーザーがダウンロードしてインストールする必要がある**公式Workflowパッケージ**として別途提供される。詳細は[インストールガイド](/installation)を参照。
:::

## 機能

- **⚙️ バックエンド管理** — ACP、Skill-Runner、Generic HTTPバックエンドタイプをサポート
- **🔧 Workflowシステム** — マルチステップの自動処理パイプラインを定義
- **📊 ダッシュボード** — タスクステータスの監視、履歴の閲覧、ログの検査
- **🖥️ サイドバーパネル** — 現在の作業コンテキストから離れずにバックエンドと対話
- **📖 組み込みMarkdownリーダー** — `.md`添付ファイルをダブルクリックしてZotero内で開く。アウトライン、検索、数式レンダリング、コードハイライト対応
- **💬 ACPチャット** — 文献をコンテキストとしたAI対話
- **🔬 Synthesis Workbench** — 文献の深層分析プラットフォーム
- **🏷️ タグ管理** — 制御されたタグ語彙と自動タグ付け
- **📈 引用グラフ** — 引用関係の可視化と分析
- **📝 トピック統合** — 自動トピック分析とレポート生成
