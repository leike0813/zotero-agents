# Home ダッシュボード

Home は Synthesis Workbench を開いたときに表示される最初のページである。ライブラリの概要、同期状態、トレンドトピックへのクイックアクセスを包括的に提供する。

![Synthesis Home ダッシュボード](/img/docs/synthesis/home.png)

## ライブラリインサイトカード

ページ上部には、Synthesis システムの現在の状態を示す統計カード群が表示される。

| 項目 | 説明 |
|------|------|
| **登録論文数** | Canonical Reference Index に含まれる論文の総数 |
| **トピック数** | 作成されたトピック統合の数 |
| **グラフノード数** | 引用グラフのノード総数（ライブラリ論文 + 外部参考文献） |
| **グラフエッジ数** | 引用グラフの引用関係の総数 |
| **同期状態** | WebDAV/Git 同期の実行状態 |

これらの指標により、ライブラリの構造化レベルと統合の進行状況を素早く把握できる。

## 同期パネル

[WebDAV Sync](webdav-sync)（推奨）または [Git Sync](git-sync)（非推奨）が設定されている場合、Home ページには同期状態パネルが表示される。

### WebDAV Sync

- **同期状態**: idle / queued / syncing / blocked_conflict / failed
- **最終同期時刻**
- **リモート HEAD 識別子**
- **アクションボタン**: 手動同期、一時停止/再開、リトライ

競合発生時には、パネルに競合の詳細とアクション選択肢（`keep_local`、`clear_after_manual_edit`）が表示される。

WebDAV 同期の詳細な設定と使い方については [WebDAV Sync](webdav-sync) を参照。

:::warning 自動同期に関する注意
WebDAV 同期の自動同期機能は十分にテストされていない。現段階では**手動同期のみを使用**し、将来のリリースで改善された後に自動同期を有効にすることを推奨する。
:::

### Git Sync（非推奨）

過去の情報については [Git Sync](git-sync) を参照。

## レビューアイテムパネル

Home ページには、保留中のレビューアイテムのクイックプレビューを表示できる。

| レビューカテゴリ | 説明 |
|-----------------|------|
| **引用マッチ** | 保留中の引用-アイテムバインディング提案 |
| **概念** | 保留中の概念、意味、別名の提案 |
| **トピックグラフ関係** | 保留中のトピック間関係 |
| **タグ提案** | 承認待ちの AI 提案タグ |

各カテゴリには保留中アイテム数がバッジで表示される。クリックすると [レビューハブ](review) の対応するサブタブに移動する。

## トレンドトピック

ページ下部には、関連論文数で並び替えられたトレンドトピックのカードリストが表示される。各カードには以下が含まれる。

- **トピック名** — クリックするとトピック詳細ページに移動
- **論文数** — トピックがカバーする論文の数
- **サマリープレビュー** — トピックの説明抜粋
- **アクションボタン** — トピックを開く、トピックを更新

アクティブなトピックが複数ある場合は、「すべて表示」リンクで Topics ページの完全なリストを閲覧できる。
