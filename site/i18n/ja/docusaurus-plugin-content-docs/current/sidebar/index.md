# サイドバー概要

## サイドバーとは

サイドバーは、Zotero Agentsが提供する便利な操作パネルで、Zoteroメインウィンドウの右側にフローティング表示されます。現在の作業コンテキストを離れることなく、バックエンドとの対話、実行状態の表示、スキル実行の管理ができます。

## 開き方

- **ツールバーボタン**: Zoteroツールバーのサイドバートグルボタンをクリック
- **メニュー**: **ツール → サイドバーを開く**
- **ダッシュボードアクション**: ダッシュボードで「サイドバーを開く/閉じる」をクリック

## アーキテクチャについて

サイドバーは**iframeアーキテクチャ**を使用しています: 3つのタブはそれぞれ独立したHTMLページを子iframeとして読み込み、postMessageを介してプラグインのメインプロセスと通信します。この設計により、タブが相互に干渉しないことが保証され、各パネルは独立したレンダリングコンテキストを持ちます。

ワークスペースモードでは、3つのタブが統合コンテナに組み込まれます。レガシーモードでは、各パネルをZoteroのライブラリペインやリーダーペインに直接埋め込むこともできます。

## 3つのタブ

| タブ | 機能 | ユースケース |
|-----|----------|-----------| 
| **ACP Chat** | 現在のアイテムをコンテキストとしてACPバックエンドと対話 | 文献読解中の質問、執筆支援 |
| **ACP Skills** | ACPバックエンド経由で実行されたスキル実行を監視・管理 | 実行進捗の表示、結果の検査、権限リクエストの処理 |
| **SkillRunner** | Skill-Runnerバックエンドの実行を表示・操作 | インタラクティブ実行の管理、認証の処理 |

## インターフェースガイド

### タブ切り替え

サイドバー上部のタブバーで3つのパネルを切り替えられます。切り替え時も前のタブの状態が保持されます。

### 幅調整

サイドバーの幅は、左境界をドラッグすることで自由に調整でき、さまざまなコンテンツ表示ニーズに対応できます。

### 共通コンポーネント

すべてのタブで以下の共通UIコンポーネントが共有されています:

- **バナー**: 上部の情報バー。現在選択されているプロジェクト情報とアクションボタンを表示
- **トランスクリプトビュー**: 対話や実行ログのメインエリア。PlainおよびBubble表示モードに対応
- **返信エリア**: メッセージや返信を送信するための下部入力エリア
- **ドロワーパネル**: 左右に展開可能な詳細パネル
- **プロンプトコンポーネント**: ユーザーの操作が必要な場合に表示されるプロンプト
- **プランコンポーネント**: 多段階プランの視覚的進捗
