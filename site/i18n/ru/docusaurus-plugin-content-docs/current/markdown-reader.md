# Встроенный Markdown-ридер

## Обзор

Плагин включает лёгкий Markdown-ридер. Когда вы **дважды щёлкаете по любому вложению `.md`** в Zotero, оно автоматически открывается во встроенном ридере, что избавляет от необходимости переключаться на внешнее приложение.

![Страница встроенного Markdown-ридера](/img/docs/markdown-reader.png)

Ридер включён по умолчанию. Чтобы отключить его (вернувшись к системному открывателю по умолчанию), снимите флажок в **Настройки → Общие**.

## Возможности

### Навигация по оглавлению

Левая боковая панель автоматически разбирает уровни заголовков (h1–h4) из документа. Нажмите на любой заголовок для быстрого перехода к соответствующему разделу.

### Полнотекстовый поиск

Поле поиска на панели инструментов поддерживает поиск по ключевым словам с подсветкой совпадений.

### Рендеринг Markdown

- **Блоки кода**: подсветка синтаксиса highlight.js для основных языков программирования
- **Математические формулы**: рендеринг KaTeX для формул LaTeX, поддержка как встроенного, так и блочного отображения
- **Таблицы, списки, цитаты**: Полная поддержка стандартного синтаксиса Markdown
- **Изображения**: Изображения с относительными путями загружаются автоматически

### Размер шрифта и ширина

- **Регулировка размера шрифта**: Регулировка от 12px до 24px; нажмите кнопки +/- на панели инструментов для пошаговой регулировки
- **Ширина чтения**: Поддержка узкого (860px) и широкого (1160px) режимов для разных размеров экрана

### Действия на панели инструментов

| Кнопка | Функция |
|--------|----------|
| Поле поиска | Полнотекстовый поиск по ключевым словам |
| Обновить | Перечитать файл и перерисовать |
| Копировать Markdown | Копировать исходное содержимое Markdown в буфер обмена |
| Копировать путь | Копировать путь к файлу в буфер обмена |
| Шрифт - | Уменьшить размер шрифта |
| Шрифт + | Увеличить размер шрифта |
| Переключить ширину | Переключение между узким/широким режимом чтения |
| Наверх | Плавная прокрутка в начало документа |
| Открыть внешне | Открыть файл системным приложением по умолчанию |

### Автоматическое оформление

Ридер автоматически адаптируется к светлой/тёмной теме Zotero без необходимости ручного переключения.

## Настройки

В **Zotero → Настройки → Zotero Agents → Общие**:

- **Включить встроенный Markdown-ридер**: Если включено, двойной клик по вложениям `.md` открывает их во встроенном ридере; если выключено, восстанавливается системный открыватель по умолчанию.

## Технические заметки

- Движок рендеринга: `markdown-it` + KaTeX + highlight.js
- Безопасность: Встроенная очистка HTML удаляет небезопасные теги и обработчики событий, такие как script/style/iframe
- Поддерживаемые типы файлов: `.md`, `.markdown` (определяются как по расширению файла, так и по MIME-типу)
