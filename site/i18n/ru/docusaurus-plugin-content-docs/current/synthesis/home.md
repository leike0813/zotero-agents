# Главная панель

Главная — это первая страница, которую вы видите при открытии Synthesis Workbench. Она предоставляет полный обзор вашей библиотеки, статус синхронизации и быстрый доступ к актуальным темам.

![Synthesis Home Dashboard](/img/docs/synthesis/home.png)

## Карточки статистики библиотеки

В верхней части страницы отображается набор статистических карточек, показывающих текущее состояние системы Synthesis:

| Метрика | Описание |
|--------|-------------|
| **Зарегистрированные статьи** | Общее количество статей, включённых в индекс канонических ссылок |
| **Количество тем** | Количество созданных синтезов тем |
| **Узлы графа** | Общее количество узлов в графе цитирований (статьи библиотеки + внешние ссылки) |
| **Рёбра графа** | Общее количество связей цитирования в графе цитирований |
| **Статус синхронизации** | Статус работы синхронизации WebDAV/Git |

Эти метрики помогают быстро оценить уровень структурированности и прогресс синтеза вашей библиотеки.

## Панель синхронизации

Если настроена [WebDAV-синхронизация](webdav-sync) (рекомендуется) или [Git-синхронизация](git-sync) (устарела), на главной странице отображается панель состояния синхронизации:

### WebDAV-синхронизация

- **Статус синхронизации**: idle / queued / syncing / blocked_conflict / failed
- **Время последней синхронизации**
- **Идентификатор удалённого HEAD**
- **Кнопки действий**: ручная синхронизация, пауза/возобновление, повтор

При возникновении конфликтов панель отображает детали конфликта и варианты действий (`keep_local`, `clear_after_manual_edit`).

Подробную информацию о настройке и использовании WebDAV-синхронизации см. в разделе [WebDAV-синхронизация](webdav-sync).

:::warning Уведомление об автосинхронизации
Функция автоматической синхронизации WebDAV-синхронизации не была тщательно протестирована. На данном этапе рекомендуется **использовать только ручную синхронизацию** и включить автоматическую синхронизацию после её доработки в будущем выпуске.
:::

### Git-синхронизация (устарела)

См. [Git-синхронизация](git-sync) для исторической справки.

## Панель элементов на рецензирование

На главной странице может отображаться быстрый обзор элементов, ожидающих рецензирования:

| Категория рецензирования | Описание |
|-----------------|-------------|
| **Совпадения цитирований** | Ожидающие предложения привязки цитирований к элементам |
| **Концепции** | Ожидающие предложения концепций, смыслов и псевдонимов |
| **Связи графа тем** | Ожидающие межтемные связи |
| **Предложения тегов** | Предложенные ИИ теги, ожидающие одобрения |

Каждая категория отображает счётчик с количеством элементов, ожидающих обработки. Нажмите для перехода на соответствующую вкладку в [центре рецензирования](review).

## Актуальные темы

В нижней части страницы отображается список карточек актуальных тем, отсортированных по количеству связанных статей. Каждая карточка содержит:

- **Название темы** — нажмите для перехода на страницу деталей темы
- **Количество статей** — число статей, охваченных темой
- **Краткое описание** — выдержка из описания темы
- **Кнопки действий** — открыть тему, обновить тему

При наличии множества активных тем используйте ссылку «Просмотреть все» для просмотра полного списка на странице тем.
