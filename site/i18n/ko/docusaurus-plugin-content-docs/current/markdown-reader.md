# 내장 마크다운 리더

## 개요

이 플러그인에는 경량 마크다운 리더가 포함되어 있습니다. Zotero에서 **`.md` 첨부를 더블클릭**하면 자동으로 내장 리더에서 열리므로 외부 애플리케이션으로 전환할 필요가 없습니다.

![Built-in Markdown Reader Page](/img/docs/markdown-reader.png)

리더는 기본적으로 활성화되어 있습니다. 비활성화하려면(시스템 기본 열림 프로그램으로 복원) **환경설정 → 일반**에서 옵션의 체크를 해제하세요.

## 기능

### 아웃라인 탐색

왼쪽 사이드바에서 문서의 제목 레벨(h1–h4)을 자동으로 파싱합니다. 제목을 클릭하면 해당 섹션으로 빠르게 이동할 수 있습니다.

### 전문 검색

도구모음의 검색 상자는 히트 하이라이팅을 지원하는 키워드 검색을 제공합니다.

### 마크다운 렌더링

- **코드 블록**: highlight.js 구문 하이라이팅으로 주요 프로그래밍 언어 지원
- **수학 공식**: KaTeX 렌더링으로 LaTeX 공식 지원, 인라인 및 블록 레벨 표시 모두 지원
- **표, 목록, 인용구**: 표준 마크다운 구문 완전 지원
- **이미지**: 상대 경로 이미지가 자동으로 로드됩니다

### 글꼴 크기 및 너비

- **글꼴 크기 조절**: 12px에서 24px까지 조절 가능; 도구모음의 +/- 버튼을 클릭하여 단계적으로 조절
- **읽기 너비**: 다양한 화면 크기에 맞게 협폭(860px) 및 광폭(1160px) 모드 지원

### 도구모음 작업

| 버튼 | 기능 |
|------|------|
| 검색 상자 | 전문 키워드 검색 |
| 새로고침 | 파일을 다시 읽고 다시 렌더링 |
| 마크다운 복사 | 원본 마크다운 콘텐츠를 클립보드에 복사 |
| 경로 복사 | 파일 경로를 클립보드에 복사 |
| 글꼴 크기 - | 글꼴 크기 축소 |
| 글꼴 크기 + | 글꼴 크기 확대 |
| 너비 전환 | 협폭/광폭 읽기 모드 전환 |
| 맨 위로 | 문서 상단으로 부드럽게 스크롤 |
| 외부 열기 | 시스템 기본 애플리케이션으로 파일 열기 |

### 자동 테마 적응

리더는 수동 전환 없이 Zotero의 라이트/다크 테마에 자동으로 적응합니다.

## 환경설정

**Zotero → 설정 → Zotero Agents → 일반**에서:

- **내장 마크다운 리더 사용**: 체크하면 `.md` 첨부를 더블클릭하여 내장 리더에서 열립니다. 체크를 해제하면 시스템 기본 열림 프로그램이 복원됩니다.

## 기술 참고사항

- 렌더링 엔진: `markdown-it` + KaTeX + highlight.js
- 보안: 내장 HTML 새니타이징으로 script/style/iframe 등 안전하지 않은 태그 및 이벤트 핸들러 제거
- 지원 파일 형식: `.md`, `.markdown` (파일 확장자 및 MIME 유형 모두로 감지)
