# ACP Chat 使用

## 功能说明

ACP Chat 允许您与配置好的 ACP 后端进行对话，对话上下文来自您当前正在查看的 Zotero 条目或阅读器中的文献。

## 使用场景

- **文献问答**：对当前阅读的论文提问，获取解释和总结
- **写作辅助**：在写作过程中获取建议
- **快速查询**：快速获取某篇文献的关键信息
- **批量处理**：对文献列表中的多个条目进行批量分析

## 界面说明

### 对话区

显示对话历史，包括用户消息和后端回复。每条对话消息包含：

- 发送者标识（用户/后端）
- 消息内容（支持 Markdown 格式）
- 时间戳

### 输入区

底部输入框支持：

- 文本输入
- 发送按钮
- 快捷键（如 Enter 发送）

### 模型/模式选择

在对话区域上方可以选择：

- **模型**：后端支持的模型列表（如 GPT-4、Claude 等，取决于后端）
- **模式**：运行模式配置
- **Reasoning Effort**：推理努力程度

## Library 上下文 vs Reader 上下文

ACP Chat 支持两种上下文模式：

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| **Library 上下文** | 基于当前在 Zotero 条目列表中选中的条目 | 浏览文库时快速查阅 |
| **Reader 上下文** | 基于当前在 Zotero Reader 中打开的文献全文 | 深入阅读时需要上下文理解 |

插件自动检测当前上下文类型并传递给后端，无需手动切换。

## 会话管理

- 对话历史自动持久化存储
- 每个后端的多个会话独立管理
- 可在 Dashboard 或侧边栏中查看历史会话

## 注意事项

- 需要先 [配置 ACP 后端](../backends/acp)
- 不同 ACP 后端的对话互不干扰
- 对话与 Zotero 条目关联，方便后续查阅
