# 侧边栏概览

## 什么是侧边栏？

侧边栏是 Zotero Skills 提供的便捷操作面板，悬浮在 Zotero 主窗口右侧。它允许您在不离开当前工作上下文的情况下与后端交互。

## 打开方式

- **工具栏按钮**：点击 Zotero 工具栏中的侧边栏切换按钮
- **菜单**：**工具 → 打开侧边栏**
- 在 Dashboard 中通过"打开/关闭侧边栏"操作

## 三个 Tab

### ACP Chat

与 ACP 后端进行对话，以当前 Zotero 条目为上下文。

- 在阅读或浏览文献时提问
- 获取文献摘要和解释
- 辅助写作和建议

→ [详细了解 ACP Chat](./acp-chat)

### ACP Skills

监控和管理通过 ACP 后端执行的技能运行：

- 查看正在运行的 ACP skill
- 查看已完成的 skil 结果
- 取消运行中的 skill

### SkillRunner

查看和交互 SkillRunner 后端的运行：

- 实时查看运行状态和输出
- 回复等待用户输入的交互式运行
- 取消运行中的任务
- 查看运行详情

## 界面说明

- 每个 tab 显示独立的 iframe 内容
- 通过顶部的 tab 栏切换
- 侧边栏宽度可调整
