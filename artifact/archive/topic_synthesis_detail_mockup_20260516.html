Synthesis Summary

Object detection is traced from Faster R-CNN's anchor-based two-stage pipeline through DETR's end-to-end set prediction shift, then through convergence fixes, real-time DETR variants, and specialized domains such as 3D detection, segmentation, underwater detection, and table structure recognition.

Core Findings

strong

DETR paradigm shift

DETR reformulates detection as direct set prediction with transformer encoder-decoder architecture and bipartite matching, removing anchors and NMS while exposing slow convergence and small-object limitations.

partial

Convergence fixes

Deformable attention, DAB-style anchor queries, DN-DETR denoising, DINO, Sparse DETR, and Group DETR form a dense cluster of 2021-2022 responses to DETR's training cost.

context

Real-time and expansion

RT-DETR, LW-DETR, RF-DETR, and DINOv3 move DETR-style detection toward real-time and edge deployment, while Mask DINO, DETR3D, underwater detection, and table recognition show domain expansion.

C1

DETR redefined object detection as end-to-end set prediction.

DETR achieved comparable COCO performance to Faster R-CNN while eliminating anchors, NMS, and hand-crafted proposal mechanisms. The tradeoff was clear: much slower convergence and weaker small-object performance.

C2

DETR's convergence problem became the core research driver.

Deformable attention, DAB-style anchor queries, denoising training, DINO, Sparse DETR, and Group DETR can be read as complementary attempts to stabilize matching and accelerate training.
