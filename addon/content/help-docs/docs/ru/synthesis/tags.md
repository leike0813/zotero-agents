# Управление тегами

## Что такое словарь тегов?

Словарь тегов (Tag Vocabulary) — это стандартизированная система тегов, используемая для единообразной аннотации литературы. В отличие от свободных тегов Zotero, теги в контролируемом словаре следуют единым соглашениям об именовании, что облегчает статистику и поиск.

## Аспекты

Каждый тег принадлежит аспекту (измерению). В настоящее время поддерживаются следующие аспекты:

| Аспект | Описание | Пример |
|-------|-------------|---------|
| `field` | Исследовательская область | `field:natural_language_processing` |
| `topic` | Исследовательская тема | `topic:transformer_architecture` |
| `method` | Исследовательский метод | `method:reinforcement_learning` |
| `model` | Использованная модель | `model:gpt-4` |
| `ai_task` | Тип задачи ИИ | `ai_task:text_summarization` |
| `data` | Набор данных | `data:imagenet` |
| `tool` | Инструмент | `tool:python` |
| `status` | Маркер статуса | `status:to_read` |

Формат тега: `^[a-z_]+:[a-zA-Z0-9/_.-]+$`, максимум 120 символов.

## Вкладка «Словарь»

На странице Synthesis Workbench → Теги → Словарь вы можете:

- **Просмотр**: Все определённые канонические теги с отображением статуса, аспекта, псевдонимов и количества использований
- **Добавление**: Создание новых канонических тегов
- **Редактирование**: Изменение метаданных тегов
- **Устаревание**: Отметить тег как устаревший, при необходимости указав тег-замену
- **Импорт JSON**: Импорт словаря тегов из JSON-файла (с предварительным просмотром перед подтверждением)
- **Экспорт JSON**: Экспорт текущего словаря в JSON-файл

Статусы тегов:
- `active`: Активный
- `deprecated`: Устаревший (имеет тег-замену)
- `warning`: Предупреждение (может потребовать проверки)

## Вкладка «Промежуточные» (ожидающие теги)

Skill **tag-regulator** автоматически анализирует метаданные литературы и генерирует предложения контролируемых тегов, отображаемые на странице «Промежуточные».

### Процесс одобрения

1. Просмотрите список предложенных тегов
2. Для каждого тега вы можете:
   - **Продвинуть**: Добавить тег в канонический словарь
   - **Отклонить**: Отклонить предложение
   - **Очистить промежуточные**: Пакетно отклонить все предложения

### Формат импорта/экспорта

Словарь тегов поддерживает импорт/экспорт в формате JSON (формат TagVocab), что позволяет:

- Миграция системы тегов между библиотеками
- Совместное использование соглашений о тегах в команде
- Резервное копирование и контроль версий

## Связанный Workflow

Стандартизация тегов и автоматический вывод управляются Workflow [Tag Regulator](#doc/workflows%2Ftag-regulator). Запуск этого Workflow может автоматически очищать и дополнять теги на основе контролируемого словаря.
