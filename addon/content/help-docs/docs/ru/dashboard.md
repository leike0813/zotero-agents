# Панель мониторинга

## Обзор

Панель мониторинга — это центральная панель управления и контроля для Zotero Agents. Здесь вы можете просматривать статус задач, управлять Workflow, просматривать историю и исследовать логи выполнения.

## Как открыть

- **Кнопка на панели инструментов**: Нажмите на значок Zotero Agents на панели инструментов Zotero
- **Меню**: **Инструменты → Открыть панель мониторинга**
- **Вкладка Zotero**: Открывается через меню, отображается как независимая вкладка Zotero

## Страницы

### Главная

Страница по умолчанию Панели мониторинга, отображающая:

- **Список Workflow**: Все доступные Workflow с кнопками запуска и настроек
- **Зона ACP Чата**: Быстрый доступ к диалогам ACP
- **Запуски навыков ACP**: Статус запусков навыков для бэкендов ACP
- **Обратная связь по навыкам**: Просмотр последних оценок и комментариев по запуску навыков
- **Сводка задач**: Обзор текущих запущенных задач

### Бэкенды

Страница управления бэкендами:

- Список всех настроенных бэкендов
- История задач каждого бэкенда
- Детальные виды бэкендов (различаются по типу)

Детальные виды бэкендов:

| Тип бэкенда | Отображение |
|-------------|---------|
| Generic HTTP | Таблица задач + логи выполнения |
| SkillRunner | Таблица запусков + область статуса + область диалога + действия ответа/отмены |
| ACP | Вид запуска навыка |

## Обратная связь по навыкам

Панель обратной связи по навыкам отображает последние отзывы о запусках навыков:

| Столбец | Описание |
|--------|-------------|
| Workflow | Название выполненного Workflow |
| Бэкенд | Бэкенд, выполнивший запуск |
| Оценка | Оценка пользователя (1–5) |
| Комментарий | Текст отзыва |
| Время | Время отправки отзыва |

Действия:
- **Фильтр**: Фильтрация по оценке, Workflow или временному диапазону
- **Экспорт**: Экспорт данных отзывов для анализа

## Статус задач

| Статус | Описание |
|--------|-------------|
| `queued` | Ожидание выполнения |
| `running` | Выполняется в данный момент |
| `waiting_user` | Ожидание ввода от пользователя |
| `waiting_auth` | Ожидание авторизации |
| `succeeded` | Выполнение успешно |
| `failed` | Выполнение завершилось с ошибкой |
| `canceled` | Отменено |

## Просмотр логов выполнения

Панель мониторинга включает встроенный просмотрщик логов:

- Фильтрация по бэкенду
- Фильтрация по Workflow
- Фильтрация по уровню лога
- Фильтрация по временному диапазону
- Экспорт диагностики
- Копирование сводки проблем

## Кнопка на панели инструментов

Кнопка-значок Zotero Agents на панели инструментов Zotero поддерживает:

- Левый клик: Открыть/переключить Панель мониторинга
- Отображает количество запущенных задач
- Показывает всплывающее окно со списком запущенных задач
