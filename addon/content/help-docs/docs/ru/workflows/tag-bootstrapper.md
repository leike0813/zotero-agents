# Tag Bootstrapper

## Назначение

Интерактивное создание контролируемого словаря тегов для исследовательской области с помощью ИИ. Рекомендуется запустить перед вашим первым [Анализом литературы](#doc/workflows%2Fliterature-analysis) для создания основы для последующей автоматической регулировки тегов.

## Сценарии использования

- Начало нового исследовательского направления и необходимость создания системы тегов
- В текущей библиотеке Zotero ещё нет контролируемого словаря тегов
- Желание, чтобы ИИ помог разработать классификацию тегов для конкретной области

## Ограничения ввода

| Тип ограничения | Описание |
|---------|------|
| Единица ввода | workflow (не нужно выбирать элементы) |
| Способ запуска | Запуск из Панели мониторинга |

## Поток выполнения

```
1. Начало взаимодействия
   └── Общение с ИИ в Панели мониторинга

2. Определение области
   └── Опишите вашу исследовательскую область и области интересов
       └── ИИ предлагает систему классификации тегов

3. Итеративное уточнение
   └── Просмотр тегов, предложенных ИИ
       └── Корректировка, добавление, удаление, переименование

4. Подтверждение и запись
   └── Запись финального словаря тегов в систему Synthesis
```

### Детали взаимодействия

- Workflow выполняется в **интерактивном** режиме, общаясь с ИИ в Панели мониторинга
- Вы можете корректировать направление в любой момент во время диалога

## Оценочная продолжительность

| Сценарий | Оценочное время |
|------|---------|
| Создание начального словаря | 3-8 минут |
| Добавление тегов | 3-5 минут |

## Рекомендации по модели

🟢 Достаточно модели средней способности; не нужна самая сильная модель.

## Выходы

После завершения выполнения контролируемый словарь тегов записывается в систему Synthesis и может быть просмотрен и управляться на странице Тегов Synthesis Workbench.

## Параметры

| Параметр | Тип | Описание | По умолчанию |
|------|------|------|--------|
| `tag_note_language` | string | Язык заметки тегов | `zh-CN` |

Доступные значения: `zh-CN`, `en-US`, `ja-JP`, `ko-KR`, `de-DE`, `fr-FR`, `es-ES`, `ru-RU`. Также поддерживается пользовательский ввод.

## Зависимости

- **Бэкенд**: Сервис Skill-Runner
- **Конфигурация бэкенда**: Настройте бэкенд типа Skill-Runner в Менеджере бэкендов
- **Навык**: Навык `tag-bootstrapper` должен быть развёрнут на Skill-Runner

## Связанные Workflow

- [Анализ литературы](#doc/workflows%2Fliterature-analysis) — Может автоматически каскадировать регулировку тегов во время анализа
- [Tag Regulator](#doc/workflows%2Ftag-regulator) — Запуск регулировки тегов на существующей литературе
