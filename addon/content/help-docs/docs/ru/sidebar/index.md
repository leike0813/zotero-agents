# Обзор боковой панели

## Что такое боковая панель?

Боковая панель — это удобная панель операций, предоставляемая Zotero Agents, плавающая на правой стороне главного окна Zotero. Она позволяет вам взаимодействовать с бэкендами, просматривать статус запусков и управлять выполнением навыков, не выходя из текущего рабочего контекста.

## Как открыть

- **Кнопка на панели инструментов**: Нажмите кнопку переключения боковой панели на панели инструментов Zotero
- **Меню**: **Инструменты → Открыть боковую панель**
- **Действие панели мониторинга**: Нажмите "Открыть/Закрыть боковую панель" в Панели мониторинга

## Заметки по архитектуре

Боковая панель использует **архитектуру iframe**: три вкладки каждая загружают независимую HTML-страницу как дочерний iframe, взаимодействуя с основным процессом плагина через postMessage. Этот дизайн гарантирует, что вкладки не мешают друг другу, при этом каждая панель имеет независимый контекст рендеринга.

В режиме рабочего пространства три вкладки интегрированы в единый контейнер; в устаревшем режиме каждая панель также может быть встроена непосредственно в панель библиотеки Zotero и панель ридера.

## Три вкладки

| Вкладка | Функция | Сценарии использования |
|-----|----------|-----------| 
| **ACP Чат** | Общение с бэкендом ACP с использованием текущего элемента в качестве контекста | Задавать вопросы во время чтения литературы, помощь в написании |
| **Навыки ACP** | Мониторинг и управление запусками навыков, выполняемых через бэкенд ACP | Просмотр прогресса выполнения, исследование результатов, обработка запросов разрешений |
| **SkillRunner** | Просмотр и взаимодействие с запусками бэкенда Skill-Runner | Управление интерактивными запусками, обработка аутентификации |

## Руководство по интерфейсу

### Переключение вкладок

Панель вкладок в верхней части боковой панели позволяет переключаться между тремя панелями. Состояние предыдущей вкладки сохраняется при переключении.

### Регулировка ширины

Ширина боковой панели может быть свободно изменена перетаскиванием левой границы для размещения различных потребностей отображения контента.

### Общие компоненты

Все вкладки имеют следующие общие компоненты UI:

- **Баннер**: Верхняя информационная панель, отображающая информацию о выбранном проекте и кнопки действий
- **Просмотр транскрипта**: Основная область для логов диалога или выполнения, поддерживающая режимы отображения Plain и Bubble
- **Область ответа**: Нижняя область ввода для отправки сообщений или ответов
- **Выдвижные панели**: Расширяемые панели деталей слева и справа
- **Компонент промпта**: Промпты отображаются, когда требуется взаимодействие с пользователем
- **Компонент плана**: Визуальный прогресс для многошаговых планов
