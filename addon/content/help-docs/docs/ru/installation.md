# Руководство по установке

## Системные требования

- **Zotero**: 7.0 или новее (рекомендуется Zotero 9)
- **Платформа**: Windows 10+, macOS 12+, Linux (x86_64 / x86 / ARM64 / ARM)

> **О версиях Zotero**: Этот плагин разработан и протестирован на Zotero 9. Zotero 8 теоретически полностью поддерживается (фреймворк плагина не имеет существенных изменений между Zotero 8/9); Zotero 7 также должен поддерживаться в теории, но не был тщательно протестирован из-за ограниченных ресурсов. Будущее обслуживание будет сосредоточено на Zotero 9. Если вы столкнулись с проблемами на Zotero 7, пожалуйста, сообщите о них в [Issues](https://github.com/leike0813/zotero-agents/issues).

## Установка плагина

### Из GitHub/Gitee Release (рекомендуется)

1. Посетите [GitHub Releases](https://github.com/leike0813/zotero-agents/releases) или [зеркало Gitee Releases](https://gitee.com/leike0813/zotero-agents/releases)
2. Скачайте последний файл `.xpi`
3. В Zotero откройте **Инструменты → Дополнения**
4. Нажмите на значок шестерёнки и выберите **Установить дополнение из файла...**
5. Выберите скачанный файл `.xpi`

### Через Zotero Plugin Marketplace

Если у вас установлен плагин [Zotero Plugin Marketplace](https://github.com/syt2/zotero-addons), вы можете найти и установить Zotero Agents напрямую из маркетплейса:

1. Нажмите на значок <img src="/img/zotero-addons_icon.png" alt="Zotero Plugin Marketplace" style="width:20px;vertical-align:middle"/> на панели инструментов Zotero, чтобы открыть маркетплейс
2. Найдите **Zotero Agents**
3. Нажмите установить

### Сборка из исходного кода

```bash
git clone https://github.com/leike0813/zotero-agents.git
cd zotero-agents
npm install
npm run build
```

Результат сборки находится в каталоге `.scaffold/build/`.

## Установка официальных пакетов Workflow

Плагин поставляется **без встроенной бизнес-логики**. Все Workflow предоставляются через отдельные официальные пакеты Workflow.

### Способ 1: Установка через меню (рекомендуется)

1. После перезапуска Zotero щёлкните правой кнопкой мыши по любому элементу → **Zotero Agents** → **📦 Установить официальные пакеты Workflow**
2. Плагин автоматически загрузит последние официальные пакеты с GitHub / Gitee
3. После завершения появится уведомление об успехе; все официальные Workflow будут видны в Панели мониторинга

### Способ 2: Установка из Настроек

1. Откройте **Zotero → Настройки → Zotero Agents**
2. В разделе **Настройки Workflow** нажмите **Установить официальные пакеты Workflow**
3. Здесь также можно переключить канал обновлений (stable / beta / dev) и проверить наличие обновлений

### Механизм обновлений

- Плагин автоматически проверяет наличие новых версий официальных пакетов при запуске
- При появлении новой версии отображается диалог подтверждения
- Список Workflow автоматически перезагружается после обновления

Репозиторий официальных пакетов Workflow: [GitHub](https://github.com/leike0813/zotero-agents-workflows) · [Зеркало на Gitee](https://gitee.com/leike0813/zotero-agents-workflows)

## Проверка установки

1. Перезапустите Zotero
2. Вы должны увидеть значок **Zotero Agents** на панели инструментов Zotero
3. Щёлкните правой кнопкой мыши по любому элементу — должно появиться подменю **Zotero Agents** (с доступными Workflow)

Если в контекстном меню отображается только пункт **📦 Установить официальные пакеты Workflow**, значит официальные пакеты ещё не установлены — следуйте инструкциям выше для их установки. После успешной установки перейдите к разделу [Начало работы](#doc/getting-started), чтобы настроить бэкенд и запустить первый Workflow.
